# Italy conditional distribution

Status: script, fixed 90/45 camera, narration, captions, target-specific
smoothing note, and public promotion approved by Jeffrey on 2026-07-18.

The episode states the population identity—integrating the conditional density
produces the conditional distribution—without implying that the fitted
distribution is obtained by numerically integrating a separately optimized
conditional-density estimate. `npcdist()` selects smoothing for its own
conditional-distribution target.

Raw screen and system-audio captures remain outside Git. This directory keeps
the exact public script, sanitized assets, editable frames, normalized
narration, accessibility files, compact proof, and approved review media.

Run `render/render_frames.sh`, `render/render_silent_animatic.sh`, and
`render/render_narrated_review.sh` to rebuild the compact review media. Run
`render/sync_public_bundle.sh --check` to test the promoted public bundle.
