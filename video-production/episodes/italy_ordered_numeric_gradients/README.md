# Italy ordered-factor and numeric gradients

Status: exact installed script, variable-type framing, visual grammar,
continuous Cove narration, captions, and Gallery promotion approved by Jeffrey

The episode gives the same displayed Italian year values two statistically
different meanings. With `year` represented as an ordered factor, `npreg()`
automatically uses ordered-kernel smoothing and reports an adjacent-level
finite difference. With `year` represented as numeric, it uses continuous
Gaussian smoothing and reports an analytic derivative.
