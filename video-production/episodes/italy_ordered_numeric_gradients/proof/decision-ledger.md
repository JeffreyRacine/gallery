## Variable-type contract

The same displayed year values receive two statistical meanings. The bundled
`Italy$year` is an ordered factor. `npreg()` therefore selects the Li–Racine
ordered kernel automatically, cross-validates its categorical smoothing
parameter, plots level-specific fitted values, and reports an adjacent-level
finite difference. When those same years are represented as numeric,
`npreg()` selects a second-order Gaussian kernel, cross-validates a continuous
bandwidth, plots a smooth fitted curve, and reports an analytic derivative.

This is not an instruction to coerce ordered variables to numeric. The user
chooses the R class that matches the predictor's meaning; `np` propagates that
choice through the analysis without an estimator switch.
