# Italy conditional quantiles

Status: script, default quantile plot, focused narration, captions,
distribution-based-method note, and public promotion approved by Jeffrey on
2026-07-18.

`npqreg()` directly obtains conditional quantiles by inverting the fitted
conditional distribution at requested probabilities. The Gallery description
distinguishes this route from the separate regression-based location-scale
method `nplsqreg()` without overloading the short.

Raw screen and system-audio captures, rejected takes, and diagnostic phonetic
experiments remain outside Git. This directory retains only the corrected
continuous narration beginning with the clean second onset, together with the
public script, sanitized sources, accessibility files, compact proof, and
approved media.

Run `render/render_frames.sh`, `render/render_silent_animatic.sh`, and
`render/render_narrated_review.sh` to rebuild the compact review media. Run
`render/sync_public_bundle.sh --check` to test the promoted public bundle.
